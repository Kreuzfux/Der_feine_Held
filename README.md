# Der feine Held

Digitaler Heldeneditor für **Das Schwarze Auge 4.0**.

> Kein offizielles DSA-/Ulisses-Produkt.

## Was der Editor kann

- Heldenbogen-Daten erfassen und bearbeiten
- OCR aus Bild oder PDF (mit Korrektur vor Übernahme)
- Mehrere Helden in einer Sitzung verwalten
- Online speichern und laden (mit Login)
- JSON-Datei importieren

## Heldeneditor benutzen

### 1) Registrieren und einloggen

1. Oben E-Mail und Passwort eingeben.
2. **Registrieren** klicken (beim ersten Mal).
3. Falls eine Bestätigungsmail kommt: Link öffnen.
4. Danach mit denselben Daten **Login**.

Im Header siehst du den Status, z. B. `Angemeldet: ...`.

### 2) Helden anlegen oder auswählen

- **Neu**: neuen Helden erstellen
- **Duplizieren**: aktive Figur kopieren
- **Löschen**: aktive Figur entfernen
- In der linken Liste auf einen Helden klicken, um ihn zu laden

### 3) Werte erfassen

Trage die Daten in den Bereichen ein:

- Stammdaten
- Eigenschaften
- Kampfwerte
- Energien
- Talente
- Inventar
- Notizen

### 4) OCR aus Scan oder PDF

1. Datei auswählen (Bild oder PDF)
2. **OCR starten**
3. Erkannte Werte im Dialog prüfen
4. **Ausgewählte übernehmen**

### 5) Online speichern / laden

- **Online speichern**: aktiven Helden in der Datenbank sichern
- **Online laden**: eigene Helden aus der Datenbank laden

## Admin-Funktion

Wenn dein Account als Admin eingerichtet ist, erscheint:

- **Admin: alle Helden laden**

Damit können alle gespeicherten Helden geladen werden.

## Sichtbarkeit der Daten

- Normale eingeloggte Nutzer sehen nur ihre eigenen Helden.
- Admin-Nutzer können zusätzlich alle Helden laden.
